# StudyFlow

StudyFlow is a web-based study planning tool designed for college students who want a simple and effective way to organize assignments, manage their time, and stay focused. The goal is to bring planning, focus, and progress tracking into one clean and easy-to-use interface.

## Features
- Add, edit, and delete study tasks
- Organize tasks by subject or deadline
- Study timer (Pomodoro-style focus tool)
- Progress tracking for completed tasks
- Responsive design for desktop and mobile
- Data saved using local storage

## Technologies
- HTML
- CSS
- JavaScript
- Local Storage
- GitHub Pages (for deployment)

## AI Usage
AI tools (such as ChatGPT) were used to help brainstorm ideas, structure the project, and generate initial code templates. All code was reviewed, understood, and modified as needed.

## Challenges
- Implementing local storage correctly
- Managing multiple pages and navigation
- Creating a clean and responsive design
- Ensuring smooth user interactions
